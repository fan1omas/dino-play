# Контроллер для google dino  

- Управляя собственной рукой, можно поиграть в динозаврика (google dino) 

- Реализуется на pyautogui и mediapipe


# Возможности

- Детекция руки в реальном времени
- Распознавание сжатия 4 пальцев
- Отслеживание прижатия большого пальца
- Эмуляция нажатий клавиш (пробела, стрелки вниз)

# Требования (!)

- Python 3.10 – 3.11 (не следует использовать версии новее 3.12, mediapipe их не поддерживает)
- MediaPipe 0.10.x
- pyautogui 0.9.x


# Установка

 1. Клонируйте репозиторий

```bash
git clone https://https://github.com/fan1omas/dino-play
cd dino-play
```

 2. Загрузите необходимые библиотеки 
```bash
pip install pyautogui mediapipe==0.10.5 
```
 3. Попробуйте поиграть:
 
- Для прыжка закройте ладонь (основные 4 пальца)

- Для приседа прижмите палец к основанию мизинца
